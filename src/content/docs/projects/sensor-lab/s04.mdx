<div class="key-terms">

<Aside type="caution" title="5V 전원 필요!">
PIR 센서는 **5V 전원**이 필요합니다. 3V3 핀에 연결하면 불안정하게 동작할 수 있으니 반드시 **VBUS(40번 핀)**에 연결하세요.
</Aside>

```python
# ── 웹서버 ──────────────────────────────────
server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
server.bind(('0.0.0.0', 80))
server.listen(3)

print(f"\n{'='*45}\n  출입 탐구 대시보드!\n  http://{my_ip}\n{'='*45}")

while True:
    try:
        # PIR 감지 체크
        if recording and pir.value() == 1:
            now_tick = time.ticks_ms()
            if time.ticks_diff(now_tick, last_detect) > 3000:  # 3초 간격
                elapsed = time.ticks_diff(now_tick, start_time) // 1000
                detect_log.append({"time": kst_str(), "sec": elapsed})
                total_count += 1
                last_detect = now_tick
                print(f"  감지 #{total_count} ({kst_str()}, +{elapsed}초)")

        server.settimeout(0.3)
        try:
            client, addr = server.accept()
        except OSError:
            continue

        request = client.recv(1024).decode()

        if 'GET /status' in request:
            elapsed = 0
            if recording and start_time > 0:
                elapsed = time.ticks_diff(time.ticks_ms(), start_time) // 1000
            body = json.dumps({
                "count": total_count,
                "pir": pir.value(),
                "recording": recording,
                "elapsed": elapsed,
                "log": detect_log[-50:],  # 최근 50개만
            })
            h = f"HTTP/1.1 200 OK\r\nContent-Type: application/json\r\nContent-Length: {len(body)}\r\nConnection: close\r\n\r\n"
            client.send(h.encode()); client.send(body.encode())

        elif 'GET /start' in request:
            recording = True
            start_time = time.ticks_ms()
            last_detect = 0
            body = json.dumps({"status": "recording"})
            h = f"HTTP/1.1 200 OK\r\nContent-Type: application/json\r\nContent-Length: {len(body)}\r\nConnection: close\r\n\r\n"
            client.send(h.encode()); client.send(body.encode())
            print("▶ 기록 시작")

        elif 'GET /stop' in request:
            recording = False
            body = json.dumps({"status": "stopped"})
            h = f"HTTP/1.1 200 OK\r\nContent-Type: application/json\r\nContent-Length: {len(body)}\r\nConnection: close\r\n\r\n"
            client.send(h.encode()); client.send(body.encode())
            print("⏹ 기록 정지")

        elif 'GET /reset' in request:
            detect_log.clear(); total_count = 0; recording = False
            body = json.dumps({"status": "reset"})
            h = f"HTTP/1.1 200 OK\r\nContent-Type: application/json\r\nContent-Length: {len(body)}\r\nConnection: close\r\n\r\n"
            client.send(h.encode()); client.send(body.encode())
            gc.collect()

        elif 'favicon' in request:
            client.close(); continue

        else:
            gc.collect(); page = make_page()
            h = f"HTTP/1.1 200 OK\r\nContent-Type: text/html; charset=utf-8\r\nContent-Length: {len(page.encode())}\r\nConnection: close\r\n\r\n"
            client.send(h.encode()); client.send(page.encode())

        client.close(); gc.collect()
    except Exception as e:
        if "ETIMEDOUT" not in str(e): print(f"에러: {e}")
        try: client.close()
        except: pass
```

### PIR 센서 특성

<Aside type="caution">
- PIR 센서는 **움직임**을 감지하지, **사람 수**를 세지 않습니다
- 2명이 동시에 지나가면 1회로 감지될 수 있습니다
- 한 명이 왔다 갔다 하면 여러 번 감지될 수 있습니다
- 센서 뒷면 **가변저항**으로 민감도와 유지 시간을 조절할 수 있습니다
- 전원 연결 후 **30초-1분** 안정화 시간이 필요합니다
</Aside>

### 흔한 문제 & 해결

| 증상 | 원인 | 해결 |
|------|------|------|
| PIR이 계속 1 출력 | 워밍업 미완료 / 민감도 과다 | 1분 대기, 가변저항 조절 |
| 감지가 안 됨 | 배선 오류 / 3.3V 연결 | VBUS(5V) 확인, OUT→GP16 확인 |
| 웹 대시보드 접속 불가 | Wi-Fi / IP 문제 | http:// 확인, 같은 네트워크 확인 |
| 감지 간격이 너무 김 | PIR 유지 시간 설정 | 센서 뒷면 가변저항(시간) 조절 |
