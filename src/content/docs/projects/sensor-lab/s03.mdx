def read_sound_level():
    """0.5초간 소리 샘플링 → 피크값 반환"""
    peak = 0
    samples = 0
    start = time.ticks_ms()
    while time.ticks_diff(time.ticks_ms(), start) < 500:
        val = adc.read_u16()
        # 중간값(32768) 기준 절대 편차
        deviation = abs(val - 32768)
        if deviation > peak:
            peak = deviation
        samples += 1
    # 0~100 스케일로 변환
    level = min(100, int(peak / 32768 * 100))
    return level, peak, samples

### 소리 센서 특성

| 항목 | 값 |
|------|------|
| 타입 | 아날로그 (ADC) |
| 핀 | GP26 (ADC0) |
| 샘플링 | 0.5초간 피크 추출 |
| 스케일 | 0~100 (상대적, dB 아님) |
| 중심값 | 32768 (무음 시 ADC 중간값) |

<Aside type="note">
Grove 소리 센서는 정밀한 dB 측정기가 아닙니다. ADC 값의 **상대적 차이**를 비교하는 용도로 사용합니다. 절대적인 데시벨 값이 필요하면 스마트폰 앱을 보조로 활용하세요.
</Aside>
