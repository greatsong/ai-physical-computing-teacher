<Aside type="tip" title="ADC란?">
**Analog-to-Digital Converter** (아날로그→디지털 변환기). Pico에서 GP26~GP28이 ADC 핀입니다. 센서의 아날로그 전압(0~3.3V)을 0~65535 사이의 숫자로 변환합니다.
</Aside>

<Aside type="caution" title="Micro USB 데이터 케이블 확인">
Pico 2 WH는 **Micro USB** 포트를 사용합니다 (USB-C가 아닙니다!). 반드시 **데이터 전송이 가능한** Micro USB 케이블을 사용하세요.
</Aside>

    lux_approx = round(raw / 65535 * 1000, 0)
    return int(lux_approx), raw

## 교사용 참고사항

### 조도 변환 참고

<Aside type="note">
Grove 조도 센서는 정확한 lux 값을 반환하지 않습니다. ADC raw값(0~65535)을 대략적 lux로 변환한 근사치입니다. 정확한 비교보다는 **상대적 차이**에 초점을 맞추세요.
</Aside>

<div class="patrol-point">
