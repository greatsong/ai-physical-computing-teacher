## 교사용 참고사항

### 차별화된 학생 지원

| 학생 유형 | 지원 방법 |
|-----------|----------|
| 아이디어가 없는 학생 | 씨앗 카드에서 가장 쉬운 것(알림/이모지) 추천 |
| 야심찬 학생 | "한 번에 다 하지 말고 하나씩" 안내, 완성도 강조 |
| 코드 충돌로 막힌 학생 | 3차시 기본 코드로 리셋 후 하나씩 추가 안내 |
| 이미 다 끝난 학생 | 친구 코드 도우미 역할 또는 발표 준비 |
