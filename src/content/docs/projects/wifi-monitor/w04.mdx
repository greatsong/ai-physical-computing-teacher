## 교사용 참고사항

### 모둠 운영 팁

<div class="patrol-point">
  <ol>
    <li>Pico 1대 + 스마트폰으로 측정 — 한 명이 Pico 들고, 한 명이 스마트폰으로 기록</li>
    <li>측정 시간이 부족하면: 3x3 격자로 축소 또는 교사가 일부 데이터 사전 입력</li>
    <li>복도/계단 등 교실 밖 측정은 시간 여유가 있을 때만</li>
    <li>소란스러울 수 있으니, 이동 시 '조용히 걷기' 규칙 안내</li>
  </ol>
</div>

### 데이터 품질

| 상황 | 대응 |
|------|------|
| 같은 위치에서 ±5dBm 변동 | 정상 — 여러 번 측정 후 평균 사용 가능 |
| 극단적 이상값 (-10 또는 -99) | 재측정 안내 |
| 빈 칸이 있는 히트맵 | 시간 내에 채우기 어려우면 빈 칸 표시도 OK |
| 모둠 간 데이터 통합 | 각 모둠이 같은 Pico에 저장하거나, 수동으로 합치기 |

### 흔한 문제 & 해결

| 증상 | 원인 | 해결 |
|------|------|------|
| /measure 페이지 안 뜸 | 이전 차시 코드 실행 중 | 4차시 코드로 교체 |
| 데이터 저장 안 됨 | 스마트폰이 다른 네트워크 | 교실 Wi-Fi 연결 확인 |
| 히트맵이 전부 같은 색 | RSSI 범위가 좁음 | 색상 범위 조정 또는 더 넓은 구역 측정 |
| Pico 꺼지면 데이터 사라짐 | 메모리에만 저장 | 측정 전 인쇄물에도 백업 기록 권장 |
