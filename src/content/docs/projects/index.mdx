## 프로젝트 목록

<CardGrid>
  <Card title="Wi-Fi 품질 모니터" icon="laptop">
    **6차시 · Pico 2 WH + Wi-Fi**

    교실 Wi-Fi 신호를 실시간 모니터링하는 웹 대시보드를 만듭니다.
    센서 없이 Pico의 내장 Wi-Fi만으로 시작하는 입문 프로젝트.

    - 1차시: 첫 웹서버 만들기
    - 2차시: 실시간 대시보드
    - 3차시: 차트와 그래프
    - 4차시: 히트맵 시각화
    - 5차시: 알림 시스템
    - 6차시: 완성과 발표

    [1차시부터 시작하기 →](/ai-physical-computing-teacher/projects/wifi-monitor/w01/)
  </Card>

  <Card title="공공데이터 전광판" icon="open-book">
    **6차시 · Pico 2 WH + API**

    나이스 급식, 기상청 날씨, 에어코리아 미세먼지, 버스 도착정보를
    하나의 전광판으로 통합합니다. 공공 API 활용 실전 프로젝트.

    - 1차시: 급식 전광판
    - 2차시: 날씨와 미세먼지
    - 3차시: 버스 도착정보
    - 4차시: 디자인 완성
    - 5차시: 나만의 API 추가
    - 6차시: 완성과 발표

    [1차시부터 시작하기 →](/ai-physical-computing-teacher/projects/public-data/p01/)
  </Card>

  <Card title="센서 실험실" icon="seti:config">
    **6차시 · Pico 2 WH + 센서 6종**

    온습도, 조도, 소리, 동작감지, CO₂, 기압 센서를
    하나씩 연결하며 배우는 센서 활용 프로젝트.

    - 1차시: 온습도 센서 (DHT20)
    - 2차시: 조도 센서
    - 3차시: 소리 센서
    - 4차시: 동작 감지 (PIR)
    - 5차시: CO₂ 센서 (SCD30)
    - 6차시: 기압 센서 (LPS22HB)

    [1차시부터 시작하기 →](/ai-physical-computing-teacher/projects/sensor-lab/s01/)
  </Card>

  <Card title="드래곤볼 레이더" icon="rocket">
    **5차시 · Pico 2 W x 15대 · 셀프 Wi-Fi**

    Pico 2W의 AP 모드로 7개의 드래곤볼을 숨기고,
    RSSI 신호 세기로 찾아내는 보물찾기 프로젝트.
    외부 인터넷 없이 셀프 Wi-Fi만으로 운영합니다.

    - 1차시: Wi-Fi의 두 얼굴 (AP vs STA)
    - 2차시: RSSI→LED 레이더 만들기
    - 3차시: 스마트폰 웹 대시보드
    - 4차시: 바이브코딩으로 7개 동시 탐지
    - 5차시: 최종 미션 — 교실 드래곤볼 헌트

    [1차시부터 시작하기 →](/ai-physical-computing-teacher/projects/dragonball-radar/d01/)
  </Card>

## 이 교재의 특징

<CardGrid stagger>
  <Card title="복사-붙여넣기 워크플로우" icon="document">
    모든 코드는 **복사해서 Pico에 바로 업로드**하는 방식입니다.
    학생이 직접 타이핑하는 부분은 AI 프롬프트뿐입니다.
  </Card>

  <Card title="FULL / LIGHT 프롬프트" icon="comment">
    모든 미션에 **FULL 프롬프트**(그대로 복사)와
    **LIGHT 프롬프트**(스스로 채우기)를 함께 제공합니다.
    학생 수준에 따라 선택할 수 있습니다.
  </Card>

  <Card title="단계별 확장" icon="star">
    Wi-Fi 모니터는 **센서 없이 Pico만으로** 시작합니다.
    센서 실험실에서 6종 센서를 하나씩 추가하며 확장합니다.
  </Card>

  <Card title="독립 운영 가능" icon="setting">
    코스 A/B와 별개로 **독립적으로 운영**할 수 있습니다.
    바이브 코딩 경험이 필요한 수업에 바로 투입 가능합니다.
  </Card>
</CardGrid>

## 공통 준비물

| 항목 | 수량 | 비고 |
|------|------|------|
| Raspberry Pi Pico 2 WH | 1 | Wi-Fi 내장 (H = 헤더 납땜 완료) |
| Micro USB 케이블 | 1 | 데이터 전송 가능한 케이블 (충전 전용 불가) |
| 노트북 | 1 | Thonny IDE 설치 |
| Wi-Fi 환경 | - | 2.4GHz 대역 (5GHz 미지원) |
| 브레드보드 + 점퍼선 | - | 센서 실험실 프로젝트에 필요 |
