## 교사용 참고사항

### 핵심 평가 포인트

<Aside type="note" title="이번 차시의 핵심은 코딩 능력이 아닌 문제 정의 역량">
"어떤 데이터가 필요한가?"를 스스로 정의하고, API를 탐색하여 적합한 것을 선택하는 과정을 관찰합니다.
</Aside>

| 평가 항목 | 관찰 기준 |
|----------|----------|
| 문제 정의 | "어떤 데이터가 필요한가?"를 스스로 정의했는가 |
| API 탐색 | 적합한 API를 탐색하고 선택하는 과정을 수행했는가 |
| AI 활용 | 프롬프트 품질이 점차 개선되는가 |
| 회복력 | 실패에서 배우는 과정이 보이는가 (에러 → AI 질문 → 수정 → 재시도) |
