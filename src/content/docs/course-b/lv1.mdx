<details class="student-content">
  <summary>학생 교재 — DHT20 기본 코드</summary>
  <div class="content">

```python
from machine import I2C, Pin
import time

i2c = I2C(1, sda=Pin(6), scl=Pin(7), freq=100000)
sensor_addr = 0x38
time.sleep(0.1)

def read_dht20():
    i2c.writeto(sensor_addr, bytes([0xAC, 0x33, 0x00]))
    time.sleep(0.08)
    data = i2c.readfrom(sensor_addr, 7)
    humidity = ((data[1] << 12) | (data[2] << 4) | (data[3] >> 4)) / 1048576 * 100
    temperature = (((data[3] & 0x0F) << 16) | (data[4] << 8) | data[5]) / 1048576 * 200 - 50
    return round(temperature, 1), round(humidity, 1)

while True:
    temp, humi = read_dht20()
    print(f"온도: {temp}°C, 습도: {humi}%")
    time.sleep(2)
```

<details class="student-content">
  <summary>학생 교재 — OLED 기본 명령어</summary>
  <div class="content">

```python
from machine import I2C, Pin
from ssd1306 import SSD1306_I2C

i2c = I2C(1, sda=Pin(6), scl=Pin(7), freq=100000)
oled = SSD1306_I2C(128, 64, i2c)

oled.fill(0)                         # 화면 지우기
oled.text("Hello!", 0, 0)            # 텍스트 표시
oled.fill_rect(0, 0, 128, 12, 1)     # 흰 사각형 (반전 헤더)
oled.text("Title", 40, 2, 0)         # 검은 글씨 (반전 헤더 위)
oled.hline(0, 13, 128, 1)            # 가로선
oled.vline(64, 14, 50, 1)            # 세로선
oled.show()                          # 화면 갱신
```

## 3차시: CSV 저장 + 테스트 + 발표 (50분)

  <p>"<code>os.stat()</code>으로 파일 크기를 확인하고, 크기가 0일 때만 헤더를 쓰는 방식을 사용합니다. 파일이 없으면 <code>except</code>로 빠져서 새로 만들고요."</p>

<details class="student-content">
  <summary>학생 교재 — CSV 저장 패턴</summary>
  <div class="content">

```python
import os

# 파일이 이미 있으면 크기 확인, 없으면 0
try:
    size = os.stat("data_log.csv")[6]
except:
    size = 0

f = open("data_log.csv", "a")
if size == 0:
    f.write("time_s,value,status\n")  # 헤더는 처음 한 번만

start = time.ticks_ms()

# 메인 루프 안에서
elapsed = time.ticks_diff(time.ticks_ms(), start) // 1000
f.write(f"{elapsed},{value},{status}\n")
f.flush()  # 즉시 저장 — 없으면 꺼질 때 데이터 손실!
```

</details>
