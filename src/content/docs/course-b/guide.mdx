코스 B는 프로젝트 기반으로, 코스 A를 마친 학생이 레벨업하며 독립적으로 진행합니다.

## 교사 역할

코스 B에서 교사는 **퍼실리테이터**입니다.

- **Lv.1 탐험가**: 센서 하나로 대시보드 완성. 교사는 센서 선택과 목표 설정을 도움.
- **Lv.2 발명가**: 복합 센서 시스템. 교사는 설계 리뷰와 디버깅 코칭.
- **Lv.3 연구자**: 장기 데이터 수집 + AI 분석. 교사는 연구 질문 설계를 도움.
- **Lv.4+ 마스터**: ML 모델 학습. 교사는 데이터 품질과 윤리 이슈를 코칭.

## 평가 전략

코스 B는 **포트폴리오 평가**를 권장합니다. 각 레벨의 산출물(대시보드, 분석 보고서, ML 모델)을 모아 포트폴리오로 구성하세요.
