각 차시 페이지에 차시별 루브릭이 포함되어 있습니다.
여기서는 **전체 수업**에 대한 종합 관찰 기록 예시를 제공합니다.

## 등급 기준

| 등급 | 기준 |
|------|------|
| **S** | 뛰어나고 놀라운 학생. 한 학교에 한두 명 수준. 교과 간 전이, 동료 지원이 자연스러움. |
| **A** | 기대 이상의 학생. 교재 범위를 넘는 자발적 탐구, 독창적 분석. |
| **B** | 이상적 학생. 핵심 역량을 체득하고 자기 언어로 표현. 최고의 배움. |
| **C** | 어느 정도 이상적인 학생. 좋은 배움. |
| **D** | 중간 정도 학생. 의미 있는 배움. |
| **E** | 조금 부족하지만 최소한의 배움은 달성. |

## 전체 종합 관찰 기록 예시

전체 15차시 종합 관찰 기록은 [학생 관찰 기록 예시](/ai-physical-computing-teacher/appendix/observation/) 페이지를 참고하세요.
