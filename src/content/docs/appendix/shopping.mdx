| 5 | **Micro USB 데이터 케이블** | — | ~3,000원 | 일반 구매 |

:::tip[I2C Hub는 언제 필요한가요?]
Grove Shield에는 I2C 포트가 **2개**(I2C0, I2C1)있습니다. 따라서:

- I2C 장치 **1~2개** → Hub 없이 각 포트에 하나씩 꽂으면 됩니다
- I2C 장치 **3개 이상** → 포트가 부족하므로 **I2C Hub 필요**

예: 4차시(DHT20 + OLED = 2개)는 Hub 불필요, 6차시(SCD30 + DHT20 + OLED = 3개)는 Hub 필요.

Hub를 I2C 포트 하나에 연결하면 최대 **5개 장치**를 동시에 사용할 수 있습니다.
:::

## I2C Hub 연결 예시

### Lv.4 프로젝트 1 (학교 공기질): 센서 4개 동시 연결

```
Grove Shield I2C1 포트
        |
   [I2C Hub 6P]
    |    |    |    |
  DHT20  SCD30  OLED  (빈 포트)
  0x38   0x61   0x3C

   + Grove Shield A2 포트
        |
     먼지 센서
      GP28
```

I2C Hub의 6개 포트 중 3개를 사용하고, 먼지 센서는 아날로그 A2 포트에 별도 연결합니다. Hub에 여유 포트 3개가 있으므로 센서 추가 확장도 가능합니다.
