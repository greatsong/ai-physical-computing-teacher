## I2C Hub 연결 예시

### Lv.4 프로젝트 1 (학교 공기질): 센서 4개 동시 연결

```
Grove Shield I2C1 포트
        |
   [I2C Hub 6P]
    |    |    |    |
  DHT20  SCD41  OLED  (빈 포트)
  0x38   0x62   0x3C

   + Grove Shield A2 포트
        |
     먼지 센서
      GP28
```

I2C Hub의 6개 포트 중 3개를 사용하고, 먼지 센서는 아날로그 A2 포트에 별도 연결합니다. Hub에 여유 포트 3개가 있으므로 센서 추가 확장도 가능합니다.
