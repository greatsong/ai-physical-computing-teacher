## 장비 준비 (팀당)

| 품목 | 수량 | 비고 |
|------|------|------|
| Raspberry Pi Pico 2 WH | 1 | 헤더 납땜 완료 모델 |
| Grove Shield for Pi Pico | 1 | v1.0 |
| USB-C 케이블 | 1 | 데이터 전송 가능한 것 |
| Grove 센서 세트 | 1 | 32종 중 수업에 사용하는 센서 |
| OLED 디스플레이 (SSD1306) | 1 | 128x64, I2C |

:::caution[중요: I2C 핀 설정]
모든 I2C 센서는 **Grove Shield의 I2C1 포트**(SDA=GP6, SCL=GP7)에 연결합니다.
코드에서는 반드시 `I2C(1, sda=Pin(6), scl=Pin(7))`로 설정하세요.
I2C0(GP8/GP9)가 아닌 **I2C1(GP6/GP7)**입니다.
:::

## 소프트웨어 설치

1. **Thonny IDE**: https://thonny.org — MicroPython 인터프리터 선택
2. **MicroPython 펌웨어**: Pico 2 WH용 최신 uf2 파일
3. **PC Python** (9차시~): Python 3.10+, pandas, matplotlib

## 학생 그룹 편성

- **권장**: 2~3인 1팀, 장비 1세트 공유
- **역할 순환**: 매 차시 코더/연결자/기록자를 교대
- **3막 팀**: 11차시부터는 4인 1팀 (센서/OLED/PC대시보드/ML)

## 차시별 사전 준비 체크리스트

### 1막 (1~4차시)
- [ ] Pico에 MicroPython 펌웨어 설치 확인
- [ ] Thonny에서 Pico 인식 확인 (COM 포트)
- [ ] DHT20, 빛 센서, OLED 작동 테스트
- [ ] Grove 케이블 여분 준비 (불량 대비)

### 2막 (5~10차시)
- [ ] 학생 AI 챗봇 접근 가능 여부 확인 (학교 네트워크 제한)
- [ ] CO2 센서(SCD41) 초기 캘리브레이션 (첫 사용 시 30분 소요)
- [ ] 심박 센서 테스트 (손가락 접촉 민감)
- [ ] PC Python 환경 설치 (9차시)
- [ ] Pico Makers 앱 접속 테스트 (https://pico-makers.vercel.app)
- [ ] Pico Buddy 앱 접속 테스트 (https://visual-pico.vercel.app)

### 3막 (11~15차시)
- [ ] Teachable Machine 접속 확인 (13차시)
- [ ] 팀 발표용 모니터/프로젝터 준비 (14차시)
- [ ] 센서 패스포트 출력물 준비 (15차시)

### 코스 B (선택)
- [ ] WiFi 환경 확인 (2.4GHz, WPA2 인증)
- [ ] 스마트폰 핫스팟 백업 준비 (학생 4~5명당 1개)
- [ ] Google 계정 준비 (Apps Script용, 학교 계정 권장)
- [ ] Flask 설치 확인: `pip install flask` (3레벨)
- [ ] Supabase 무료 계정 생성 (4레벨)
- [ ] Pico Makers 앱에서 코스 B 모드 정상 동작 확인
