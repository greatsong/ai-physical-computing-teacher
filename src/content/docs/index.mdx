이 지도서는 학생용 교재 **"AI 피지컬 컴퓨팅"**과 동일한 구조로 구성되어 있습니다.
각 차시 페이지에서 학생이 보는 내용과 함께, **교사 스크립트**, **예상 학생 반응**, **지도 팁**, **평가 기준**을 확인할 수 있습니다.

## 지도서 구성 요소

<CardGrid stagger>
  <Card title="교사 스크립트" icon="document">
    도입(5분) → 전개(35분) → 정리(10분) 구간별로
    **교사가 말할 문장**을 구체적으로 제시합니다.
    자신의 스타일에 맞게 변형하여 사용하세요.
  </Card>

  <Card title="예상 학생 반응" icon="comment">
    올바른 반응과 오개념 반응을 구분하여 제시합니다.
    **"학생이 이렇게 말하면 이렇게 대응하세요"** 형태로
    현장에서 바로 활용할 수 있습니다.
  </Card>

  <Card title="지도 팁" icon="star">
    시간 배분, 순회 지도 포인트, 자주 발생하는 오류와 대응법,
    빠른 학생/느린 학생 차별화 전략을 포함합니다.
  </Card>

  <Card title="평가 루브릭" icon="list-format">
    차시별 S~E 등급의 관찰 기록 예시와
    성취기준 체크리스트를 제공합니다.
    생활기록부 작성에 직접 활용할 수 있습니다.
  </Card>
</CardGrid>

## 코스 선택

같은 장비, 같은 센서를 사용하지만 수업 운영 방식이 다릅니다.
학교 상황과 학생 수준에 맞는 코스를 선택하세요.

<CardGrid>
  <Card title="코스 A: 15차시 수업" icon="open-book">
    **매주 50분, 3막 구성으로 단계별 성장**

    - **1막** — 센서와 친해지기 (4차시)
    - **2막** — 바이브 코딩으로 확장 (6차시)
    - **3막** — 대시보드와 머신러닝 (5차시)

    직접 코딩 → 바이브 코딩 → ML 프로젝트까지,
    15주에 걸쳐 교사 주도로 체계적으로 진행합니다.

    [코스 A 1차시 지도안으로 이동 →](/ai-physical-computing-teacher/course-a/act1/lesson-01/)
  </Card>

  <Card title="코스 B: 프로젝트 레벨업" icon="rocket">
    **3시간씩, 레벨업하며 만들기**

    - **Lv.1 탐험가** — 센서 1개 대시보드
    - **Lv.2 발명가** — 복합 센서 스마트 시스템
    - **Lv.3 연구자** — 바이브 코딩 + AI 분석
    - **Lv.4 마스터** — 머신러닝 + 실제 문제 해결

    학생 자기주도 프로젝트 중심,
    교사는 퍼실리테이터로 레벨별 지도 전략을 활용합니다.

    [코스 B 지도 가이드로 이동 →](/ai-physical-computing-teacher/course-b/guide/)
  </Card>
</CardGrid>

## 수업 전체 흐름 (코스 A)

| 단계 | 1막 (1~4차시) | 2막 (5~10차시) | 3막 (11~15차시) |
|------|--------------|----------------|-----------------|
| **코딩 방식** | 직접 코딩 | 바이브 코딩 (AI 활용) | 바이브 코딩 + ML |
| **교사 역할** | 시범 → 따라하기 지도 | 프롬프트 작성 코칭 | 프로젝트 퍼실리테이터 |
| **핵심 관찰** | 코드-동작 대응 이해 | AI 코드 비판적 검토 | 설계-구현-발표 전 과정 |
| **주요 위험** | 포트/배선 혼동 | AI 코드 맹신 | 팀 내 역할 불균형 |
